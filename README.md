# KOMDEX-BJC Bestel Tool

Mobiele besteltool voor BJC artikelen.

## GitHub Pages Setup

### Stap 1: Repository aanmaken
1. Ga naar [GitHub.com](https://github.com) en log in
2. Klik op het **+** icoon rechtsboven â†’ **New repository**
3. Geef het een naam (bijv. `komdex-bestel-tool`)
4. Kies **Public** (of Private als je betaalt)
5. Vink **NIET** "Add a README file" aan (we hebben er al een)
6. Klik **Create repository**

### Stap 2: Bestanden uploaden
1. In je nieuwe repository, klik op **uploading an existing file**
2. Sleep `index.html` en `README.md` naar het upload gebied
3. Scroll naar beneden en klik **Commit changes**

### Stap 3: GitHub Pages activeren
1. Ga naar **Settings** (bovenaan in je repository)
2. Scroll naar **Pages** (in het linker menu)
3. Bij **Source**, kies **Deploy from a branch**
4. Kies branch: **main** (of **master**)
5. Kies folder: **/ (root)**
6. Klik **Save**

### Stap 4: Je website bekijken
- Na een paar minuten is je site live op:
- `https://jouwgebruikersnaam.github.io/repository-naam/`
- (Vervang `jouwgebruikersnaam` en `repository-naam` met je eigen gegevens)

## Bestanden updaten
- Upload gewoon een nieuwe versie van `index.html` naar GitHub
- De site wordt automatisch bijgewerkt binnen een paar minuten
